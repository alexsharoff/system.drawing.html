The HtmlRenderer Class

System.Drawing.Html.HtmlRenderer

The Html renderer is a static class that contains this two methods for drawing HTML:

Render(Graphics g, string html, PointF location, float width)
Renders HTML on the specified device, by filling down using the specified width starting from the specified location.

Render(Graphics g, string html, RectangleF area, bool clip)
Renders HTML on the specified device, by filling down using the specified width starting from the specified location. If clip is set to true, it will only draw on the specified area.

This class may have more functionality in future releases.
