HtmlPanel

System.Windows.Forms.HtmlPanel

You can use it just as an HTML container, just place the panel on a form and feed the Text property with HTML code.

Try the Sample Form to see another panel.

Properties

  • AutoScroll Gives the panel scroll bars when content needs it.
  • HtmlContainer Gets the object that manages HTML rendering.
  • Text Gets or sets the HTML source for rendering.
  • MeasureBounds() Re-measures bounds of HTML content. It should be not necessary to be called in regular use.

Differences between HtmlPanel and HtmlLabel

  • HtmlLabel should be used for short Labelish fragments
  • HtmlLabel default font is the Operative System's default font for controls, HtmlPanel default font is the operative system's default Serif font.
  • HtmlLabel may have transparent background
  • HtmlLabel may be autosized
  • HtmlPanel has better performance
